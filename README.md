> [!IMPORTANT]
> **DO NOT use the "Download ZIP" button!**
> This project uses **Git LFS** (Large File Storage) for high-quality assets. Downloading the repository as a ZIP file will result in broken assets (missing models/textures). Please **clone** the repository using a Git client to ensure all files are downloaded correctly.
