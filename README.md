## Aisle-0: Toy Store Anomaly
    "The toys are watching. Can you spot the difference before the cycle resets?"

Aisle-0 is an atmospheric, anomaly-detection horror game set in the unsettlingly familiar corridors of a liminal toy store. As a night-shift security guard, you are trapped in a loop where your only weapon is your memory and attention to detail.

## 🕹️ Core Gameplay: The Loop
Reality is not as stable as it seems in the toy store.

Observation: Walk through the aisles and memorize every shelf, shadow, and sound.

The Decision: If everything looks normal, proceed forward. If you spot an anomaly—no matter how subtle—turn back immediately.

Survival: Each correct choice brings you closer to the exit, while a single mistake resets the cycle.

## 🛠️ Technical Implementation
Procedural Anomaly System: Engineered a robust randomized logic in C# to manage environmental state changes, ensuring a unique experience in every run.

Modular Clean Code: Architected using scalable and maintainable scripts to handle complex game states and anomaly triggers efficiently.

High-Fidelity Atmosphere: Integrated custom Blender models with optimized baked lighting and spatial audio in Unity to heighten environmental storytelling.
